# **App Name**: AFTDEO Manager

## Core Features:

- User Authentication: Secure user login and authentication to protect sensitive data. This will leverage Firebase Authentication for ease of use.
- Attendance Tracking: Log employee attendance with clock-in and clock-out functionality.
- B3 Waste Management: Admin-only module to track and manage B3 waste data, including input, storage, and disposal.
- Dashboard Overview: Provides a high-level overview of key data points such as attendance, B3 waste levels, and important contacts.
- Daily Report Generator: Generate daily operational reports automatically, summarizing key activities and data points. Use AI to incorporate salient points into the report and act as a tool to avoid overlooking important information.

## Style Guidelines:

- Primary color: Pertamina Blue (#0076A9) to maintain brand identity and instill trust.
- Background color: Light gray (#F0F4F8) for a clean and professional look.
- Accent color: Teal (#26A69A) for interactive elements and highlights.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Use Lucide Icons for a consistent and clean visual style.
- Use a grid-based layout for a responsive and organized dashboard.
- Subtle transitions and animations to enhance user experience (e.g., card hover effects).
